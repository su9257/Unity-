> **2. 更改字体，让1（数字）和I（i）或者和l（L）看着如此相似**

> **3. Info和Data这种其实表示一个意思的东西用多种命名（在Lua中要他们的命）**

>

> 
> **9. 在lua中写了一个函数含有参数A，但参数A从来不用，在函数内部通过其他手段获取A并使用，迷惑调用者为什么传参了但是不生效**
